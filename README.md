# MOSINT 2025

Энэ төсөл нь Facebook-ийн олон нийтийн мэдээллийг (OSINT) хайж, шинжлэхэд зориулагдсан энгийн вэб хэрэглүүр юм. Flask ашиглан бүтээгдсэн бөгөөд PostgreSQL өгөгдлийн сантай холбогдон профайл, пост, сэтгэгдэл, урвал, 8 оронтой дугаарын мэдээллийг уншиж харуулдаг. Доорх зааврыг дагаснаар шинэхэн хэрэглэгч ч хэдхэн алхмаар ажиллуулж туршиж болно.

## Урьдчилсан шаардлага
- Python 3.10+ суусан байх
- PostgreSQL сервер бэлэн байх (унших эрхтэй хэрэглэгч)
- `psycopg2` (эсвэл `psycopg2-binary`) зэрэг шаардлагатай Python сангууд

## Суурилуулалт
1. Эх кодыг татаж авна (`git clone` эсвэл ZIP-ээр).
2. Төслийн хавтас руу орно:
   ```bash
   cd mosint2025
   ```
3. Сангуудыг суулгана (виртуал орчин ашиглавал тохиромжтой):
   ```bash
   pip install -r requirements.txt  # requirements.txt байхгүй бол Flask, psycopg2-г тусад нь суулгана
   ```

## Орчны хувьсагч (аймгууд) тохируулах
PostgreSQL холболт болон нууц түлхүүрээ дараах хувьсагчаар удирдана. `.env` файл эсвэл таны терминалд export хийж болно.
- `MOSINT_DB_NAME` – Өгөгдлийн сангийн нэр
- `MOSINT_DB_USER` – Хэрэглэгчийн нэр
- `MOSINT_DB_PASSWORD` – Нууц үг (жишээ утга бүү хатуу кодлоорой)
- `MOSINT_DB_HOST` – Серверийн хаяг (жишээ нь `localhost`)
- `MOSINT_DB_PORT` – Порт (анхдагч нь `5432`)
- `MOSINT_SECRET_KEY` – Flask session-ийн нууц түлхүүр (өгөгдөлгүй бол санамсаргүйгээр үүснэ)

## Аппликейшн ажиллуулах
1. Орчны хувьсагчаа тохируулсны дараа хөгжүүлэлтийн серверийг асаана:
   ```bash
   flask run
   ```
   эсвэл
   ```bash
   python app.py
   ```
2. Хөтөчөөр `http://127.0.0.1:5000` руу орно. Анх удаа нэвтрэх үед `admin` / `admin` нэр, нууц үгийг ашиглан демо байдлаар нэвтрэнэ.
3. Амжилттай нэвтэрсний дараа хайлтын хуудсанд UID эсвэл 8 оронтой утас оруулж хайлт хийнэ. Хоёр талбарыг зэрэг бөглөхөөс зайлсхий.

## Ашиглалтын товч заавар
- **UID-гаар хайх:** Profiles, posts, comments, reactions, fbphone хүснэгтээс тохирох бүх мөрийг татаж харуулна. Пост, сэтгэгдэл, урвалын тоон товчоог болон урвалын төрлийн статистикийг хүснэгтээр үзүүлнэ.
- **8 оронтой утсаар хайх:** `fbphone` хүснэгтийн `eightdigitnumbers` талбараар шууд шүүнэ. Зөвхөн 8 цифртэй, зайгүй тоо бичнэ.
- **Аюулгүй байдал:** Код нь зөвхөн унших/шинжилгээний зориулалттай. Жинхэнэ орчинд өөрийн нууц үг, түлхүүрийг орчны хувьсагчаар солихоо мартуузай.

## Нэмэлт зөвлөмж
- Серверийг интернетээр ил гаргах бол HTTPS, галт хана зэрэг хамгаалалтын нэмэлт тохиргоо хий.
- Өгөгдлийн сангийн эрхийг зөвхөн унших хандалтаар хязгаарлах нь тохиромжтой.
- Хэрэв багц суулгах үед `psycopg2` алдаа өгвөл PostgreSQL хөгжүүлэлтийн багцыг (`libpq-dev` гэх мэт) суулгаад дахин оролдоорой.

Эдгээр алхмыг дагаснаар MOSINT 2025-ыг шууд ажиллуулж, өөрийн өгөгдлийн сантай туршиж үзэх боломжтой.
